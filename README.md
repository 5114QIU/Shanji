# ⚡ 闪记

> 你的个人观影记忆库

一款面向个人使用的轻量化观影记录工具。通过极简的交互设计与结构化数据管理，提供高效的记录、存储与检索服务，形成用户数字化的个人观影记忆库。

![License](https://img.shields.io/badge/license-MIT-blue.svg)

## ✨ 核心功能

### 📥 影视录入
- **TMDB 搜索录入** - 一键搜索全球影视数据库，自动填充海报、剧名、演员、年份等元数据
- **搜索结果筛选** - 按"电影"或"电视剧"分类筛选搜索结果
- **手工录入兜底** - 对于搜索无果的内容，支持手动上传图片和填写信息

### ✍️ 极速记录体系
- **结构化评分** - 提供按照喜爱程度点亮爱心功能
- **标签选择** - 提供"二刷、神作、致郁"等快捷标签，支持用户自定义
- **感悟记录** - 支持纯文本记录观影心情，不限字数

### � 档案库管理
- **记录流展示** - 首页按照观看状态展示，支持按类型/年份/喜爱程度排序筛选
- **全局搜索** - 支持按标题/演员名搜索已存记录
- **详情页二次编辑** - 支持对记录进行修改或删除
- **感悟管理** - 支持添加、编辑、删除观影感悟

### 📊 数据可视化
- **观影统计看板** - 自动生成观影数量、偏好类型、评分分布等统计图表
- **观影总览** - 统计观影数量、总时长、连续观影天数
- **偏好画像** - 类型占比可视化展示
- **时间趋势** - 月度折线图显示观影趋势

### 🎨 分享与增长
- **精美海报生成** - 将影片海报与经典台词合成宣传风格海报
- **台词管理** - 支持经典台词选择、自定义添加和编辑
- **字体选择** - 提供多种字体样式供选择
- **海报切换** - 支持多张剧照切换

### 👤 基础设置
- **账户管理** - 支持头像/昵称修改、密码重置、退出登录
- **数据安全** - 云端存储，数据不丢失

## 🛠️ 技术栈

| 类别 | 技术 |
|------|------|
| 前端框架 | React 19 + TypeScript |
| 构建工具 | Vite 6 |
| 后端服务 | Supabase (数据库 + 认证 + Storage) |
| CSS 框架 | Tailwind CSS 4 |
| 动画 | Motion |
| 图片生成 | html-to-image |
| 影视数据 | TMDB API |

## 📱 适合谁用？

- 🎭 **追剧达人** - 记录追剧进度，管理片单
- 🎬 **电影爱好者** - 写下观影感想，收藏好片
- 📸 **分享达人** - 生成精美海报，分享观影心情
- 📝 **生活记录者** - 用影视记录生活点滴
